# Memo CLI

A simple and efficient notes manager for your terminal, built with Node.js.

## Installation

1. Clone the repository:

```bash
git clone https://github.com/hungtsou/memo-CLI.git
cd memo-CLI
```

2. Install dependencies:

```bash
npm install
```

3. Link the package globally to use the `memo` command:

```bash
npm link
```

## Usage

After linking, you can use the `memo` command from anywhere in your terminal:

```bash
memo
```

## Development

The project uses:

- **Node.js** - Runtime environment
- **yargs** - Command-line argument parsing
